# Welcome
	ในปัจจุบันมีเทคโนโลยีเข้ามามีบทบาทสำคัญในการดำเนินชีวิต ทั้งนี้ทางผู้จัดทำก็ได้เล็งเห็นการประยุกต์ใช้ การควบคุมระบบด้วย Microcontroller โดยนำมาเป็นตัวประมวลผล สั่งการให้มีการรดน้ำพืช ด้วยการพิจารณาจากความชื้นที่มีอยู่ในดินหรือปริมาณฝนที่ได้รับ และเมื่อค่าความชื้นในดินลดน้อยลงกว่าเกณฑ์ที่ตั้งไว้ ก็จะทำให้ Microcontroller สั่งการให้ระบบทำงานเพื่อส่งจ่ายน้ำไปยังต้นไม้นั้นเอง และเมื่อทำการส่งจ่ายน้ำได้ถึงตามปริมาณที่ได้ตั้งค่าไว้แล้ว Microcontroller ก็สั่งการให้หยุดจ่ายน้ำ
